# 🛒 Blinkit Retail Sales Analytics Dashboard

## 📌 Project Overview

The Blinkit Retail Sales Analytics Dashboard is an interactive data analytics project developed using **Power BI**, **MySQL**, **SQL**, **Excel**, and **DAX**. The project analyzes retail sales data to provide insights into business performance through interactive visualizations and key performance indicators (KPIs).

The dashboard enables users to explore sales performance across different categories, brands, cities, and payment modes using dynamic filters and slicers, helping identify sales trends and business opportunities.

---

## 🎯 Project Objectives

- Analyze overall retail sales performance.
- Monitor key business metrics using KPI cards.
- Compare sales across categories, brands, and cities.
- Analyze revenue, profit, and order performance.
- Provide an interactive dashboard for business analysis.

---

## 🛠️ Technologies Used

- Power BI
- MySQL
- SQL
- Microsoft Excel
- DAX

---

## 📊 Dashboard Features

The dashboard includes the following business metrics:

- Total Revenue
- Total Profit
- Total Orders
- Average Order Value
- Average Profit per Order
- Profit Margin (%)

Interactive visualizations include:

- Revenue by Category
- Revenue by Brand
- Revenue by City
- Revenue by Payment Mode
- Monthly Sales Trend
- Interactive Slicers for Category, City, and Payment Mode

---

## 🗄️ SQL Analysis

The SQL analysis was performed using MySQL and includes:

- Data retrieval using SELECT statements
- Aggregate functions (SUM, AVG, MAX, MIN, COUNT)
- GROUP BY and ORDER BY operations
- LIMIT clause
- Subqueries
- Window Functions (RANK)

Business queries include:

- Total Revenue Analysis
- Total Profit Analysis
- Revenue by Category
- Revenue by Brand
- Revenue by City
- Highest Revenue Categories
- Loyalty Customer Analysis
